Hardware

The motherboard bus connects components of a computer, such as the CPU, and connects it to the motherboard. A bus is the path through which a device sends its data so that it can communicate with the CPU and/or other devices.

The CPU is the brain of the computer. CPU is the most important element of a computer system. The CPU is responsible for interpreting and executing most of the commands from the computer's hardware and software.

RAM is an acronym for random access memory. RAM is the most common type of memory found in computers and printers. RAM is the computer component that boots your computer. RAM acts as an intermediate memory between the hard drive and CPU. We use RAM because RAM acts as a high speed storage for data the computer is currently working on.

The hard drive is the place where everything is stored. If your hard drive is damaged you can lose all your of your documents, pictures, music, and videos.Hard drives were invented over 50 years ago and have been used in personal computers since the mid-1980s. Hard drives are efficient computer memory devices that uses simple magnetism to store information.

The motherboard is a printed circuit board that is the foundation of a computer, located at the bottom of the computer case. It distributes power to the CPU, RAM, and all other computer hardware components. Most importantly, there is a 'bus' located in the motherboard that allows hardware components to communicate with one another.

The monitor shows you graphic images as you work or play. The monitor also displays your cursor (pointing device) and shows where it is and what the mouse is hovering over. The monitor is the visual interface that allows computer users to see open programs and use applications, such as Web browsers and software programs.

Keyboards are one of the primary input devices. A keyoard is like a set of typewriter-like keys that enable you to enter data into other devices and into a computer. Christopher Sholes was an American inventor who invented the first practical typewriter and the QWERTY keyboard.

A mouse is a input device that is mostly used with desktop computers and sometimes laptops if you aren't certain on how to use the touch pad. A mouse is a pointing device that detects two-dimensional motion relative to a surface. There are two types of mice and they work differently, either using the rolling rubber ball(in a ball-type mouse) or by bouncing a light off your desk(in an optical mouse).
