HTML

HTML stands for Hypertext Makeup Language, it's a standardize system for tagging text files to achieve font, color, etc. HTML is used to make web pages. HTML was created by Tim Berenrs-Lee in 1989.The language is written in the form of HTML elements consisting of tags enclosed in angle brackets (like html). Browsers do not display the HTML tags and scripts, but use them to interpret the content of the page. HTML allows images and objects to be embedded and can be used to create interactive forms. HTML consists of a series of short codes typed into a text-file by the site author.

HTML Tags

HTML is written with normal old text. What you use to set certain sections apart as headings, subtitles, bold text, underlined text, etc is a series of what we call "tags". Tags are what seperate normal text from HTML code. The tags themselves don't appear when you view your page through a browser, but their effects do. You write tags with < and > which enables you to custamize your web page how ever you want to.

Attributes

Attribute selectors select an element using the presence of a given attribute or attribute value. You can change attributes by inserting differenet tags. Some examples of attributs are the title attribute, the href attribute, and the size attributes. The attribute selectors can be useful for styling forms without class or ID.

Describing 4 Diffferent Tags

"!--...--" defines a comment, "div" defines a section in a document, "ol" defines an ordered list, and "strong" defines an important text. There is also h1, h2, h3, h4, h5, and h6. This makes it so you can size your title from h1 being the biggest, to h6 being the smallest.
